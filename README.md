# Typecho Plugin VOID

> ✏ 一款简洁优雅的 Typecho 主题的插件

作为计算机术语时，VOID 的意思是「无类型」。

## 使用方法

下载本插件，**解压后将文件夹名称修改为 VOID**，上传至插件目录启用。

请保证插件下 cache 目录可写！不要与 ExSearch 或 PandaBangumi 插件同时开启！

配合 [VOID 主题](https://github.com/monsterxcn/Typecho-Theme-VOID) 使用。

## 特性

 - 浏览量统计
 - 文章点赞
 - 文章字数统计
 - 评论投票与自动折叠
 - 访客互动展示
